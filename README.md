# Product Aggregator Platform

A full-stack web application designed to aggregate, search, and filter products efficiently. This project utilizes a client-server architecture to deliver a seamless user experience, leveraging **MongoDB Atlas Search** for high-performance, fuzzy text search capabilities.

- **User Accounts**: User profile management including avatar uploads.
- **Responsive UI**: Modern frontend interface for browsing and managing products.
